# 解法
問題文の指示通りに0.12345...の小数点以下を文字列の結合により構成し，指定された桁の掛け算を行えばよい．  
1から1000000までの整数を結合してできる小数は明らかに小数桁以下が1000000桁以上となるから，i=1000000まで結合を行えば十分である．

``` C++
#include <iostream>
using namespace std;


int main() {
    string S = "";
    for (int i = 1; i <= 1000000; i++) {
        S += to_string(i);
    }
    cout << (S[0] - '0') * (S[9] - '0') * (S[99] - '0') * (S[999] - '0') * (S[9999] - '0') * (S[99999] - '0') * (S[999999] - '0') << endl;
}
```
